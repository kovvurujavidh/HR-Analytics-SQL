# HR Analytics using SQL

## Project Overview
This project analyzes an HR dataset containing 1,470 employee records using SQL to solve real-world HR business problems.

## Tools Used
- MySQL
- SQL

## Dataset
- IBM HR Analytics Dataset
- 1,470 Employee Records

## Business Questions Solved
- Total Employees
- Department-wise Employee Count
- Attrition Count & Rate
- Average Salary by Department
- Highest & Lowest Salary
- Overtime Analysis
- Years at Company Analysis
- Job Role Analysis
- Gender Analysis
- Marital Status Analysis
- Employee Retention Analysis

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- COUNT()
- AVG()
- MIN()
- MAX()
- SUM()
- CASE
- Subqueries
- LIMIT

## Key Insights
- Research & Development has the largest workforce.
- Attrition rate is approximately 16%.
- Salary distribution varies across departments and job roles.
- Overtime and years at company provide useful retention insights.
