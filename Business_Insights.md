# Project Summary

This project analyzed **1,470 employee records** using **MySQL** to answer **30 HR business questions**. SQL concepts such as **SELECT, WHERE, GROUP BY, ORDER BY, HAVING, Aggregate Functions, Subqueries, CASE statements, and LIMIT** were applied to generate actionable HR insights for workforce analysis and business decision-making.
